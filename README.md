# Folha de Pagamento - Sistema de Gestão

Um software moderno e intuitivo desenvolvido para facilitar o gerenciamento de folhas de pagamento em empresas de todos os tamanhos.

## 🎯 Objetivo

Simplificar e automatizar o processo de cálculo e gestão de salários, descontos e benefícios dos funcionários, economizando tempo e reduzindo erros manuais.

## 💻 Tecnologias Utilizadas

- **HTML** - Estrutura e marcação das páginas
- **CSS** - Estilização e design responsivo
- **JavaScript** - Lógica da aplicação e interatividade

## ✨ Funcionalidades (Planejadas)

- [ ] Cadastro de funcionários
- [ ] Cálculo automático de salários
- [ ] Gestão de descontos (INSS, IR, etc.)
- [ ] Relatórios de folha de pagamento
- [ ] Exportação de dados em PDF
- [ ] Interface amigável e responsiva

## 🚀 Como Começar

### Pré-requisitos
- Um navegador web moderno (Chrome, Firefox, Safari, Edge)
- Editor de código (VS Code, Sublime, etc.) - opcional

### Instalação

1. Clone o repositório:
```bash
git clone https://github.com/Edisoncarlosdefaria/primeiro_trabalho.git
```

2. Abra o arquivo `index.html` no seu navegador
